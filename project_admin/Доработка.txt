БД
1. Создать корректную модель БД: 
	-внешние ключи с _id, 
	-создание связей между объектами
	-constraints
	
2. Создание методов для объектов модели:
	-выборка связанных элементов через relationships
	-пересмотр сервисов создания данных и заполнения форм из объектов БД: максимально использовать relationships

3. Переделать загрузку справочников:
	-массовая загрузка - использовать общую сессию для корректной последовательности сохранения объектов
		предусмотреть полное удаление всех справочников перед загрузкой
	-одиночные справочники - использовать не удаление, а перезапись

4. Во всех операциях записи в БД предусмотреть обработку исключений

5. Продумать подготовку фэйковых тестовых данных
